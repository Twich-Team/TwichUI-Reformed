### Version 0.0.2-ALPHA [ March 27 2026 ]

#### BugFixes

- Fixed an issue where the Mythic+ timer would not properly count all minions
- Fixed a bug that was causing buttons to appear on the chat frame
- Fixed an issue where the chat message entries would not size properly within dungeons

#### New

- Migrated all remaining notification configurations into the central Notifications section
- Moved away from an ElvUI mover to anchor the notification to an in-house mover; added chat frame attachment options
- Added the ability to reset global appearance settings to default
- Introduced a setup wizard for onboarding
- Re-skinned buttons that are available on notifications to the in-house skin
- Revamp of global Appearances, including the ability to reset, global font, status texture, mute sound, class icon picker, etc.
